  ## **Requisitos Funcionais**

**CRUD**
  
 - RF01 - Cadastro de Livros: O sistema deve permitir o cadastro de novos livros, recebendo dados como Título, Autor, Ano de Publicação e Gênero.  
 - RF02 - Listagem de Livros: O sistema deve exibir uma lista de todos os livros cadastrados na base de dados.
 - RF03 - Atualização de Dados: O sistema deve permitir a edição das informações de um livro cadastrado.
 - RF04 - Exclusão de Livros: O sistema deve permitir a remoão de um livro do catálogo.

    ## Próximas evoluções
    
    - RF05 - Gestão de Usuários: O sistema deve permitir a criação de contas de usuário para que cada pessoa gerencie sua própria biblioteca.
    - RF06 - O sistema deve conter páginas para cada usuário exibir suas listas de livros, como um perfil. 
    - RF07 - Os usuários devem poder se comunicar com um chat ou fórum.
